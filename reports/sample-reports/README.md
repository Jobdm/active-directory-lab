# Sample Active Directory Documentation Report

This directory contains example reports generated by the automated documentation system.

## Report Types

- **Domain-Info.csv** - Domain and forest configuration
- **OUs.csv** - Organizational unit structure
- **Users.csv** - User account inventory
- **Groups.csv** - Security group inventory
- **GPOs.csv** - Group Policy Object inventory

## Report Generation

Reports are automatically generated using:
\\\powershell
& ".\scripts\powershell\Generate-ADDocumentation.ps1"
\\\

**Note**: Actual reports contain sensitive information and are excluded from version control.
